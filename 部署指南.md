# 🚀 恒市值法智能理财助手 — 部署指南

## 方式一：GitHub Pages（推荐，免费）

### 步骤 1：创建 GitHub 仓库
1. 打开 https://github.com/new
2. Repository name 填写：`investment-advisor`（或你喜欢的名称）
3. 选择 **Public**（公开）
4. **不要勾选** "Add a README file"（因为项目已有文件）
5. 点击 "Create repository"

### 步骤 2：推送代码
在项目目录下执行以下命令（替换 `<你的GitHub用户名>`）：

```bash
# 设置远程仓库地址
git remote add origin https://github.com/<你的GitHub用户名>/investment-advisor.git

# 推送代码
git push -u origin master
```

### 步骤 3：开启 GitHub Pages
1. 进入仓库页面 → 点击 **Settings**
2. 左侧菜单 → **Pages**
3. "Branch" 下拉选择 `master`，文件夹选 `/ (root)`
4. 点击 **Save**
5. 等待 1-2 分钟，页面会显示公网地址：
   ```
   https://<你的GitHub用户名>.github.io/investment-advisor/
   ```

### 步骤 4：测试
在浏览器打开上述地址，确认网站正常显示（三个主题切换、滑块回测、图表渲染）。

## 方式二：Vercel（推荐备选，速度更快）

1. 打开 https://vercel.com → 用 GitHub 账号登录
2. 点击 "New Project" → 导入你的 GitHub 仓库
3. 无需任何配置，直接点击 **Deploy**
4. 获得地址如：`https://investment-advisor.vercel.app`

---

## 方式三：Netlify

1. 打开 https://app.netlify.com → 用 GitHub 账号登录
2. 点击 "Add new site" → "Import an existing project"
3. 选择 GitHub 仓库，直接 Deploy
4. 获得地址如：`https://xxx.netlify.app`

---

## 注意事项

- **Excel 下载链接**：网站中的下载按钮指向 `/workspace/恒市值法理财操作表（稳健型）.xlsx`，部署到 GitHub Pages 后需要把这个文件也上传到仓库根目录，并修改 `index.html` 中的下载链接为相对路径 `恒市值法理财操作表（稳健型）.xlsx`
- **数据文件较大**：`js/data.js` 约 57KB，首次加载可能需要 1-2 秒
- **主题切换**：URL 参数 `?theme=business|modern|tech` 可指定默认主题
